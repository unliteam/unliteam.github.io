Список доступных команд

🏧 Персонализация​

!name - редактировать ваше первоначальное имя, экспортированное из Steam. Это позволяет изменить ваше отображаемое имя на сервере
nicknames_enabled 0/1 (defaults to 1)

!nickcolor r,g,b - статически изменить цвет вашего ника. Цвет указывается в формате RGB (красный, зеленый, синий), где r, g, b - значения от 0 до 255.
Пример использования: !nickcolor 255,0,0

!status - добавить текстовый статус к вашему нику на сервере. Можно также добавить цвет к статусу, используя атрибут <c=r,g,b>.
Пример использования с цветом: !status <c=255,0,0> Sample text

🎱 Геймплей​

!slbreturn or hold down R - перемещает вас на стоянку
!small - уменьшает игрока в масштабе
/lol - LOL!
!stopmusic / !startmusic - отключает / возобновляет проигрывание музыкального сопровождения в игре
/fire - устраняет все возгорания
!camera - получить камеру / возовель / попкорн
/y - крикни что-нибудь
!discord/wiki - открыть соответствующий ресурс
!slbstuck or hold down G - возвращает вас на чекпоинт

🔲 Консоль​

u1sound.volume значение - громкость звукового сопровождения и по совместительству включение и отключение
u1sound.next.ambient - воспроизвести следующую композицию

⌨ Горячие клавиши​

F1 - дополнительные настройки
ALT + E - сесть на какой-либо объект
F6 - taunt меню
F11 - включить ps1 эффект
F - переключает режим камеры от первого лица к виду от третьего лица

🔬 Форматирование текста в EasyChat​

С помощью этих тегов разметки можно форматировать текст в чате на нашем сервере и любом другом, на котором предустановлен исходный easychat.​

СинтаксисОписание
<stop>Останавливает выполнения предыдущего тега. Пример использования: <color=255,0,0>Hello <stop>World!
<color=>Статически изменить цвет текста. Цвет указывается в формате RGB (красный, зеленый, синий), где r, g, b - значения от 0 до 255. Пример использования: <color=255,0,0>Red!
<c=>Статически изменить цвет текста. Цвет указывается в формате HEX. Пример использования: <c=00ff00>green text!
<hsv=>Цветовая модуляция с учетом значений HSV. Остальные переменные здесь. Пример использования: <hsv=t()*300>Rainbow text
<bhsv=>Цветовая модуляция со значениями HSV на фоне текста. Пример использования: <bhsv=t()*300>Rainbow background
<flash=>Вспышка. Цветовая модуляция в формате RGB. Пример использования: <flash=0,255,255>Cyan flashing text
<hscan>Горизонтальный эффект сканирования текста. Пример использования: <hscan=3,0,255,0>green scan
<vscan>Вертикальный эффект сканирования текста. Пример использования: <vscan=3,0,255,0>green scan
<rotate=>Вращение текста по углу. Пример использования: <rotate=t()*300>zoom zoom I rotate
<zrotate=>Вращение по осям рыскания и крена. Пример использования: <zrotate=t()*300>zoom zoom I rotate
<texture=>Демонстрация текстуры. Пример использования: <texture=matsys_regressiontest/background,64>
<translate=>Переводит текст из его исходного положения в другое. Пример использования: <translate=rand()*10,rand()*10>Im angry!
^0-15Предварительно запрограммированные цвета готовые к использованию. Пример использования: ^13this is red
<wrong>Пометить текст как "неправильный". Пример использования: <wrong>This text is wrong
<background=>Рисует фон тексту. Пример использования: <background=0,255,255>Cyan background
&..Цвета из Minecraft, основанные на цвете карата. Пример использования: &ethis is yellow
<font=>Изменить шрифт. Список частично поддерживающих шрифтов. Пример использования: <font=DermaLarge>I am big!

🧩 Серверный вывод данных​

uptime - время непрерывного функционирования сервера
loss - потеря интернет пакетов в процентном соотношении
status- вывод информации об присоединившихся игроках
srvviewer - общий сбор информации о сервере
